MATH
```c
int main(){
	int x = 1;
	int y = 15;
	float z = 0;
	
	//arithmetic operators: + - = ** * / % ++ -- 
	
	//add
	z = x + y; ;
	//subtract 
	z= y - z;
	//multiply
	z= x * y;
	 
	return 0;
}
```
data types matter
`/` divides, be wary of data types
`**` exponents 
`%` returns the remainder
`++/--` increment and decrement

# Understanding Network Hacks (Ballmann)

Bastian Ballmann's 187-page "Attack and Defense with Python"
(Springer, 2015). Hands-on protocol-level hacking: every attack ships with a
short Python script and a matching detection/defense variant.

## Coverage
- Networking fundamentals (OSI, Ethernet, ARP, IP, TCP/UDP, firewalls, MITM).
- Layer 2, TCP/IP, DNS, HTTP, WiFi, and Bluetooth attacks using raw sockets
  and Scapy.
- Appendix: Scapy reference.

## Where it fits
This is the fundamentals + code layer. Built on [[scapy]] and
[[python-sockets]]. Concrete techniques: [[arp-spoofing]], [[port-scanning]],
[[dns-spoofing]], [[sql-injection]], [[wifi-attacks]], [[bluetooth-attacks]].

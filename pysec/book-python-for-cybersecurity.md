# Python for Cybersecurity

279-page book mapping the MITRE ATT&CK framework to executable
Python tradecraft. Each chapter covers one ATT&CK tactic and walks two concrete
techniques with working tooling, ending in a Summary and Suggested Exercises.

## Coverage
- 13 tactic chapters: Pre-ATT&CK, Initial Access, Execution, Persistence,
  Privilege Escalation, Defense Evasion, Credential Access, Discovery,
  Lateral Movement, Collection, Command & Control, Exfiltration, Impact.
- Windows-centric: Registry autoruns, WMI, Event logs, logon types.
